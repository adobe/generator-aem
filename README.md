# AEM Project Generator

More to come but to get started:

1. Install Yeoman 

```
$ npm -g install yo
```

2. Clone this repo
3. Link this generator so it can be found

```
$ npm link
```

4. Run the generator 

```
$ yo @adobe/aem
```
